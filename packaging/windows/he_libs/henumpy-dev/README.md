## Hello HENumpy!

Numerical computation is the foundation of modern scientific and engineering research, covering a wide range of fields from differential equations to linear algebra operations. These calculations are essential for simulation, optimization, data analysis and decision making. NumPy is one of the most popular numerical computation libraries in the Python ecosystem, widely used for array operations, linear algebra, random number generation and other tasks. 

In some applications of privacy computing, dense state data structures are insufficient for performance needs, such as in areas such as financial modeling and large-scale data processing that require more advanced numerical tools. 

Our fully dense state numerical computation package HE-Numpy is designed to fill this gap, providing users with a NumPy-based extension that focuses on high-performance fully dense state numerical computations. It can easily handle fully dense state large-scale data sets, high-dimensional tensors and complex linear algebra operations.
