# hp.polyfit

多项式拟合。对给定的采样点数据进行最小二乘法多项式拟合。

## 签名

```python
hp.polyfit(x, y, deg)
```

## 参数

- `x`: 数组密文 — 形状(M,)，M 个采样点的 x 坐标
- `y`: 数组密文 — 形状(M,)，M 个采样点的 y 坐标
- `deg`: int — 拟合多项式的最高次数

## 返回值

数组密文 — 形状(deg+1,)，多项式系数，最高幂优先。

## 示例

```python
import henumpy as hp
import crypto_toolkit as ct
import numpy as np

hp.initDict()
ct.initSK()

x = ct.encrypt(np.array([0.5, 0.3, 4.3, 0.1]))
y = ct.encrypt(np.array([2.1, 4.0, 5.2, 40.5]))
res = hp.polyfit(x, y, 3)
print(ct.decrypt(res))
# 输出 [-102.36215538  524.62593981 -379.04329572   73.26043233]
```

## 精度验证

**测试一**：拟合多项式 `y = 2x³ + 3x² + 4x + 5`，100 个采样点 (x ∈ [0, 5])

| | 三次项 | 二次项 | 一次项 | 常数项 |
|---|---|---|---|---|
| HENumpy | 2.0000000000098 | 3.0000000000223 | 3.9999999999815 | 5.0000000000067 |
| Numpy | 2.0000000000000 | 3.0000000000000 | 4.0000000000000 | 5.0000000000001 |
| 精确值 | 2 | 3 | 4 | 5 |

对比精确值，HENumpy 可保证 11 位有效数字。

**测试二**：拟合 cos(x) + ±0.01 噪声，HENumpy 与 Numpy 结果有效位数 13 位。
