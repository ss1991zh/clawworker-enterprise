# transEncType

### 行列加密方式转换

## 参数

数组密文

`A`：输入元素，$ m\times n $维数组

## 返回值

数组密文 `transEncType`：转换$ A $的加密方式

## 示例
```python
import henumpy as hp
import crypto_toolkit as ct
import numpy as np

hp.initDict()
ct.initSK()

# 行加密 → 行加密
AA = np.array([[ 1.,  2.,  3.],[ 2., -3.,  4.],[ 3.,  1.,  4.]])
A = ct.encrypt(AA)
res = A.transEncType()			
hp.printCipher(res)
# 输出 [[4.796843320511944e-246, 5.058335908245208e+226, -4.028841656015545e-217],
#       [              3.0,               3.0,               3.0],
#       [0.2116924932272934, 0.4182207687688583, 0.6702668697628127],
#       [0.4233849864545873, -0.627331153153287, 0.8936891596837507],
#       [0.6350774796818808, 0.2091103843844292, 0.8936891596837511]]

# 列加密 → 行加密
BB = np.array([[ 0.5,  4.,  2.],[ 4., 5.,  -6.],[ -0.1,  0.7,  2.2]])
B = ct.encrypt(BB, encrypt_by_column=True)
res = B.transEncType()		
hp.printCipher(res) 
# 输出 [[6.369913979134091e+197, 3, 0.10043078496593874, 0.8034462797275099, 0.40172313986375474],
#       [1.223660006959814e-125, 3, 0.8538605256085926, 1.0673256570107414, -1.2807907884128895],
#       [2.0392749893845974e+159, 3, -0.02078049141974997, 0.1454634399382497, 0.45717081123449915]]
```
