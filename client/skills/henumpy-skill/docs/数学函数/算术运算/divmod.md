# divmod

### 逐元素同时返回商和余数

## 参数

标量密文或数组密文

- `x_1`：被除数
- `x_2`：除数 如果 $ x_1.shape \neq x_2.shape $，则将它们广播到通用形状（成为输出的形状）。
- `output_encrypt_type`：标量明文，可选 输出的数组密文的加密方式，默认返回的加密方式与输入数组的加密方式一致，若设定 $ output\_encrypt\_type=0 $，则返回的数组为行加密形式；若 $ output\_encrypt\_type=1 $，则返回的为列加密形式。

## 返回值

元组密文 $ divmod(x_1,\ x_2)=(out1,\ out2) $ `out1`：数组密文，$ x_1/x_2 $的商，$ floor\_divide(x_1,\ x_2) $ `out2`：数组密文，$ x_1/x_2 $的余数，$ mod(x_1,\ x_2) $

## 示例
```python
import henumpy as hp
import crypto_toolkit as ct
import numpy as np

hp.initDict()
ct.initSK()

# 标量和标量
x1 = ct.encrypt(5)
x2 = ct.encrypt(3)
res = hp.divmod(x1, x2)	
print(ct.decrypt(res))
# 输出  (array(1.), array(2.))

# 向量和向量
aa = np.array([0.5, 0.3, 4.3, 0.1])
a = ct.encrypt(aa)
bb = np.array([2.1, 4.0, 5.2, 40.5])
b = ct.encrypt(bb)
res = hp.divmod(b, a)		
print(ct.decrypt(res))
# 输出 (array([  4.,  13.,   1., 405.]), array([1.00000000e-01, 1.00000000e-01, 9.00000000e-01, 4.33091344e-14]))

# 标量和向量
res = hp.divmod(x1, a)
print(ct.decrypt(res))
# 输出 (array([10., 16.,  1., 50.]), array([1.09901915e-14, 2.00000000e-01, 7.00000000e-01, 1.09901915e-14]))

# 标量密文和标量明文
res = hp.divmod(x1, 2.0)  	
print(ct.decrypt(res))
# 输出 (array(2.), array(1.))

# 向量密文和标量明文
res = hp.divmod(b, 2.0)
print(ct.decrypt(res))
# 输出 (array([ 1.,  2.,  2., 20.]), array([1.00000000e-01, 5.15283525e-15, 1.20000000e+00, 5.00000000e-01]))
    
# 数组和数组
AA = np.array([[ 1.,  2.,  3.],[ 2., -3.,  4.],[ 3.,  1.,  4.]])
A = ct.encrypt(AA)
BB = np.array([[ 0.5,  4.,  2.],[ 4., 5.,  -6.],[ -0.1,  0.7,  2.2]])
B = ct.encrypt(BB)
res = hp.divmod(A, B)     
print(ct.decrypt(res))
# 输出 (array([[ 2.00000000e+00, -4.70509429e-16,  1.00000000e+00],
#              [-3.58150576e-16, -1.00000000e+00, -1.00000000e+00],
#              [-3.10000000e+01,  1.00000000e+00,  1.00000000e+00]]), 
#      array([[ 7.92532226e-16,  2.00000000e+00,  1.00000000e+00],
#             [ 2.00000000e+00,  2.00000000e+00, -2.00000000e+00],
#             [-1.00000000e-01,  3.00000000e-01,  1.80000000e+00]]))

# 标量和数组		
res = hp.divmod(x1, B)		
print(ct.decrypt(res))
# 输出  (array([[ 10.,   1.,   2.],
#               [  1.,   1.,  -1.],
#               [-51.,   7.,   2.]]), 
#       array([[ 0.00000000e+00,  1.00000000e+00,  1.00000000e+00],
#              [ 1.00000000e+00,  1.98946898e-15, -1.00000000e+00],
#              [-1.00000000e-01,  1.00000000e-01,  6.00000000e-01]]))

# 向量和数组
aa2 = np.array([0.5, 0.3])
a2 = ct.encrypt(aa2)
CC = np.array([[ 1.,  2.],[ 2., -3.]])
C = ct.encrypt(CC)
res = hp.divmod(a2, C)			
print(ct.decrypt(res))
# 输出 (array([[ 1.45404716e-16,  2.90809431e-16],
#              [-1.92334817e-16, -1.00000000e+00]]), 
#       array([[ 0.5,  0.3],
#              [ 0.5, -2.7]]))

# 数组和明文
res = hp.divmod(A, 4)		
print(ct.decrypt(res))
# 输出 (array([[-1.16421059e-15, -3.42414878e-16, -9.58761660e-16],
#              [-6.95185455e-17, -1.00000000e+00,  1.00000000e+00],
#              [-2.68083074e-16, -1.13935306e-15,  1.00000000e+00]]), 
#      array([[1.00000000e+00, 2.00000000e+00, 3.00000000e+00],
#             [2.00000000e+00, 1.00000000e+00, 9.39891239e-16],
#             [3.00000000e+00, 1.00000000e+00, 9.14292710e-15]]))
```
